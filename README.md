# 💰 Finance Tracker

A personal finance management web application built with **Django** that helps users securely manage their income, expenses, and savings goals.

## ✨ Features

- User Registration & Login
- Secure Authentication
- Add Income & Expense Transactions
- View Transaction History
- Create Savings Goals
- User-specific Data Management
- Responsive UI with Tailwind CSS

## 🛠️ Tech Stack

- Python
- Django
- SQLite
- HTML
- Tailwind CSS

## 🚀 Installation

```bash
git clone https://github.com/Sarthak210805/FinanceTracker.git
cd FinanceTracker

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate
python manage.py runserver
```

Visit:

```
http://127.0.0.1:8000/
```
